# 题外话
初衷:在学习jsDoc 之前先来谢谢如何写好github的README.md文件，毕竟这是个门面，知道一些常规写法不为过。

# 1.标题
  # 一级标题
  ## 二级标题
  ### 三级标题
 tips: #和文字之间需要有一个空格
 
# 2.字体样式 -- 粗体斜体
  *斜体*
  **粗体**
  ***粗体斜体***
 tips:一对*代表斜体，2对*代表粗体，3对*代表粗体加斜体
 
 
 # 3.代码块写法

  ```
   (async(ctx,next)=>{
    awiat xxxx;
   })();
  ```
